일반 모드는 현재 운영 파일을 그대로 엽니다. 테스트 모드는 별도 파일과 별도 저장소를 사용해 리팩터링, 하드코딩 제거, 회귀 테스트를 안전하게 검증합니다.

일반 모드

기존 `index.html`, `script.js`, `style.css`를 그대로 사용하는 현재 앱입니다.

테스트 모드

테스트 모드 열기
